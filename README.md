# Frontend Course Project

A web application built as part of a university frontend development course.

## Technologies
- JavaScript
- HTML
- CSS

## Project Description
The application focuses on frontend functionality, user interaction and basic application logic.  
The project was implemented individually as part of coursework.

## What I Learned
- Basics of frontend application structure
- Handling user interactions
- Writing clearer and more maintainable frontend code
- Working with version control (Git)

## How to run
1. Clone the repository:
```bash 
 git clone https://github.com/erikalankinen/Frontend-course-project.git
```
2. Open the project in Visual Studio Code
3. Run using a local development server or open the index file in a browser
